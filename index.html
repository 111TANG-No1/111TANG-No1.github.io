PROJECT 01
Yonsei Course Finder
延世大学选课决策工具 · 独立产品设计与开发
OCR Claude API JavaScript Python GitHub Pages
背景与痛点
  • 延世大学选课靠手速抢课，考核信息全藏于各课程独立PDF，需逐一打开大量无标题页面比对
  • 严重影响"保奖学金 + 控考试压力"双目标下的选课决策效率——这是我自己真实遇到的问题
产品决策与行动
  • 设计OCR管道抽取课程考核占比（期中/期末/作业/出勤等），结构化输出为可筛选网页
  • 设计override底座机制：OCR不稳定时手动数据自动兜底，确保工具始终可用（MVP思维）
  • Ease Score模型：将期中+期末占比换算为0–100直观指标，用户无需手动计算即可判断课程压力
  • 集成Claude API自然语言搜索，支持口语化需求筛选（如"不想有团队项目"）
  • 设计本地fallback机制，AI不可用时自动降级为关键词匹配，保证演示稳定
  • 覆盖多学院35门课程，支持学院/专业/考核类型三级联动筛选
